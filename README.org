* NormalmapGenerator
This is a fork of the NormalmapGenerator GUI application. The Qt GUI was replaced with a simple CLI that allows the same option but convenient usage in batch scripts.

This was mostly done to automate certain tasks in Morrowind modding. Some future functions might be added that serve no other purpose than to make this application more useful for that purpose.
